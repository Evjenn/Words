# Cities Game (Гра в міста)

## Опис проєкту

Це десктопна гра "Міста", реалізована на Java.  
Гравець та комп’ютер по черзі називають міста. Кожне наступне місто має починатися на останню літеру попереднього.

---

## Як запустити

### Через .jar файл

1. Переконайтесь, що у вас встановлена Java (версія 17+).
2. Відкрийте термінал або командний рядок.
3. Виконайте команду: java -jar cities-game-1.0.jar.

---

## Як грати

1. Введіть назву міста у поле вводу.
2. Натисніть кнопку "Зробити хід".
3. Комп’ютер зробить свій хід у відповідь.
4. Наступне місто має починатися з останньої літери попереднього.

---

## Правила

- не можна повторювати міста
- вводьте міста по останній букві
- перемагає той, хто назвав міста з більшею кількістью літер
- вибір режиму гри за кількістю ходів
- враховується як хід гравця, так і хід комп'ютера.

---

## Використані технології

- Java 17 / 21
- Swing (графічний інтерфейс)
- Gradle (система збірки)
- Checkstyle (контроль якості коду)
- ООП (розділення логіки: GameService, GameState, UI)
